# Boolean类型

## Boolean()转型函数

| 数据类型  | 转换为true的值    | 转换为false的值 |
| --------- | ----------------- | --------------- |
| Boolean   | true              | false           |
| String    | 非空字符串        | ''              |
| Number    | 非0（包括无穷大） | 0 和NaN         |
| Object    | 任何对象          | null            |
| undefined | n/a               | undefined       |
